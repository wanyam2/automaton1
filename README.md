## Automaton : 초음파로 연주하기

## 작품 제작에 사용한 센서와 엑추에이터, 역할
1. 초음파 센서 - 물체와 거리를 측정하는 데에 쓰임
2. 부저 - 멜로디를 재생하는 데에 쓰임
